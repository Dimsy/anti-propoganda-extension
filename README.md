## So how can i use it?
1. Clone the repo
```
git clone https://github.com/Dimsy/anti-propoganda-extension.git
```

2. Open Chrome and go to `chrome://extensions/`
3. Click `Load unpacked extension`
4. Select folder with the cloned repo
